# User Guide

## How to Use
1. Open PowerShell
2. Navigate to project folder
3. Run: .\automation_runner.ps1

## Menu Options
- Option 1: Run validation demo
- Option 2: Create sample data
- Option 3: View reports
- Option 4: Manage portfolio
- Option 5: View documentation
- Option 6: Push to GitHub
- Option 7: Open in VS Code
- Option 8: System information
- Option 9: Exit

## Detailed Instructions

### Option 1: Data Validation Demo
Run the Python validation script on sample data.
Outputs cleaned CSV and JSON reports.

### Option 2: Sample Dataset
Create test data with various scenarios.
Useful for demonstrations and testing.

### Option 3: View Reports
Browse generated files in Output folder.
Preview CSV and JSON content.

### Option 4: Portfolio Assets
Create documents for your application.
Generate case studies and summaries.

### Option 5: Documentation
Access this guide and technical docs.

### Option 6: GitHub
Prepare project for version control.
Get instructions for pushing to GitHub.

### Option 7: VS Code
Open project in code editor.
Edit and explore files.

### Option 8: System Info
View project statistics and details.

### Option 9: Exit
Close the automation system.

## Troubleshooting
- If scripts don't run: Set-ExecutionPolicy RemoteSigned
- If Python not found: Install Python and add to PATH
- If missing packages: pip install pandas numpy
